If you are adding a custom domain, read this!!! (important)

Create a CNAME record in the user settings of the domain registrar you have purchased your domain from.

Type Name / Host Value / Content TTL
CNAME @ www.{{ $.ContentDomain }} auto (or 3600 seconds/1 hour)

Validate that the CNAME record has been added successfully by visiting https://dnschecker.org/#CNAME and entering your domain. The results should show www.{{ $.ContentDomain }}.

If you have only just created it, please wait for a few minutes for it to take effect.
